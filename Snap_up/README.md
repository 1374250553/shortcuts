# 超级抢购工具

## 功能说明

* 实现多电商平台的链接跳转，复制电商链接，点击抢购捷径自动跳转到相应的购买页面，提高你的抢购效率。
* 一般抢购步骤：复制商品链接》打开「京东」电商app》打开客服对话框》粘贴商品链接》发送商品链接》对话框点击商品链接跳转抢购页面
* 超级抢购工具步骤：复制商品链接》打开超级抢购工具》自动跳转抢购页面
